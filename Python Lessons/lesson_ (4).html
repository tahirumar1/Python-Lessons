🚌Bus analogy for // and %: 27 students, buses hold 8 each. How many full buses? 27 // 8 = 3. How many students left over? 27 % 8 = 3. Draw this on the board — it sticks.
Operator precedence — PEMDAS / BODMAS
PParentheses
EExponents **
MD* / // %
AS+ and −

Within the same level, Python evaluates left to right. When in doubt — use parentheses. They always win.

main.py — trace these on paper first
print(2 + 3 * 4)        # → 14  (not 20 — * before +)
print((2 + 3) * 4)      # → 20  (parentheses first)
print(2 ** 3 ** 2)       # → 512 (right-to-left: 3**2=9, 2**9=512)
print(10 / 2 * 5)        # → 25.0 (left-to-right: 5.0 * 5)
print(10 % 3 + 2 ** 2)   # → 5   (** first, then %, then +)
💡Teaching tip: Have students trace line 3 on paper — most will predict 64 (left to right). The right-to-left behaviour of ** surprises everyone and cements why PEMDAS matters.

Example 2 — Rounding and f-string format specifiers
main.py
result = 10 / 3

print(result)                    # 3.3333333333333335 — ugly
print(round(result, 2))           # 3.33 — better
print(f"{result:.2f}")            # 3.33 — f-string way
print(f"{result:.4f}")            # 3.3333
print(f"{result:.0f}")            # 3 — rounded to whole number

# Tip: use augmented assignment to update variables
score = 10
score += 5    # same as score = score + 5
score *= 2    # same as score = score * 2

Example 3 — The math module
main.py
import math

print(math.pi)                   # 3.141592653589793
print(math.sqrt(144))            # 12.0
print(math.floor(3.9))           # 3  — always rounds down
print(math.ceil(3.1))            # 4  — always rounds up
print(math.pow(2, 10))           # 1024.0

Activity 1 — Tip calculator
Activity 1 · 10 minutes · Individual
Build a restaurant tip calculator
A real-world program that uses input, float conversion, arithmetic, rounding, and f-string formatting all together. Every student has used or seen a tip calculator — this makes the lesson tangible.
  1. Ask for the bill amount and the tip percentage using float(input(...))
  2. Calculate the tip amount: bill * (tip_percent / 100)
  3. Calculate the total: bill + tip
  4. Print all values rounded to 2 decimal places using :.2f in f-strings

Key vocabulary for this lesson
Floor division (//)Divides and rounds down to the nearest whole number. Always returns an int if both operands are int.
Modulo (%)Returns the remainder after division. Key use: checking even/odd (n % 2 == 0).
Operator precedenceThe rules that determine which operators run first. Same as PEMDAS/BODMAS from maths class.
Augmented assignmentShortcuts like +=, -=, *= — update a variable in place.
ModuleA file of pre-written Python code you can import and use. import math gives access to maths functions.
Format specifier (:.2f)Inside an f-string, controls how a float is displayed. :.2f = 2 decimal places.

Teacher notes
⚠️Common mistake: Students using ^ for exponents (from calculator habit). In Python, ^ is the bitwise XOR operator — it will silently give a wrong answer. Emphasise: exponent in Python is **.
⚠️Common mistake: Forgetting that / always returns a float — 9 / 3 gives 3.0, not 3. Students are often surprised. Reinforce: use // when you need an integer result.
⚠️Common mistake: Writing {value.2f} instead of {value:.2f} — the colon is required. A missing colon gives a NameError or AttributeError.
💡Pacing note: If time is short, skip the math module demo and assign it as homework exploration. The operators and precedence content is essential; math is a nice-to-have for Day 4.
